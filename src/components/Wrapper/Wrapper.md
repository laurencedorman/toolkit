```js
<Wrapper>
  <h3>Simple Wrapper with display Flex and tag HTML type</h3>
  <ul>
    <li>choose <b>content direction</b> with props "<b>direction</b>" (column by default)</li>
    <li>choose <b>tag type</b> of wrapper "<b>div || header || footer</b>" (div by default)</li>
  </ul>
</Wrapper>

</Wrapper>
```

<Wrapper type="footer">

### Installation

  + Go to your package.json and add in dependencies:
  
  `"mano-toolkit": "git+ssh://git@bitbucket.org:mano_mano/toolkit.manomano-lan.com.git"`
  
  + Next in your terminal execute:
  
  `yarn install`
  
<br/>

### Utilisation

  `import { component } from 'mano-toolkit';`
